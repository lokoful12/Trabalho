Programação web responsiva

Tags de estilo físico

São tags que descrevem o signficado de seu conteúdo, porém não especificam a forma como o texto deverá ser apresentado.

  • negrito
    • Função para transformar a frase, ou palavra em negrito
  • itálico
    • Transformar em itálico a frase
  • Monoespaçado
    • Estilo máquina de escrever
  • Big
    • Aumenta o tamanho da fonte
  • Small
    • Diminui o tamanho da fonte
  • Sub
    • Subscrito
  • Sup
    • Sobrescrito
  • U
    • Sublinhado
  • S
    • Tracejado

Tags de Estilo Lógico

São tags que descrevem o significado de seu conteúdo, porém não especificam a forma como o texto deverá ser apresentado.

  • EM
    • Enfatizar
  • Strong
    • Mais enfatizado que EM
  • Code
    • Identifica amostra de códigos e exibe em fonte de extensão fixa.
  • Samp
    • Identifica texto de extensão fixa, como Code, porém não usada para identificação de código.
  • DFN
    • Indica uma definição
  • Cite
    • Indica citação

Textos Pré-Formatado

Por padrão no html o texto só sofre alteração por conta das tags, exceto pela tag pré, que faz uma exceção à regra, utilizada para códigos simples.

Thematic Break

A tag hr e br, não possuem tag de fechamento e devem ser definidas no arquivo CSS.

Listas

  • Lista Ordenadas
    • Se apresentam classificadas, normalmente, com números.
  • Listas não ordenadas
    • Apresentam Símbolos e não são classificadas.
  • Listas de Glossário
    • Cada item da lista tem um termo e uma definição.
